    Java 쥬니어 개발자 모임 아해가 2019 공개세미나를 합니다.

    핵심주제는 대용량 처리

    이번 세미나의 핵심주제로 대용량 처리입니다. 최근 많은 서비스기업에서 요구하는 대용량 처리 관련 다양한 관점의 기술을 공유합니다

    이외에 Java과 Spring의 이슈들을 함께 다루고 있습니다

    장소

    https://www.hanbit.co.kr/rent/location.html

    지하철 이용시 공항철도, 2호선 홍대입구역 하차 후 4번 출구 또는 5번 출구에서 5분 소요. 자동차 이용시 ① 합정역/홍대입구역 방향에서 오실 때 : 홍대입구역에서 신촌역 방면으로 가는 동교동 삼거리에서 신호 대기하였다가(2차선 좌회선 차선을 이용), 좌회전 신호시 유시티불가마 건물 옆골목 오르막 길로 50m 올라와서 우회전. ② 신촌역 방향에서 오실 때 : 신촌역에서 홍대입구역 방면으로 900m 직진 후, 동교동 삼거리를 가기 전 유시티불가마 건물 옆골목에서 1시 방향에 있는 오르막 길로 50m 올라와서 우회전.
